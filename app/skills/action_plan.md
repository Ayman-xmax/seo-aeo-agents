# Skill: Action Plan (Phase-1 close)

## Mission
Turn the analysis into a clear, approvable plan. This is the LAST thing the user sees in
Phase 1 — make it decision-ready.

## Format
1. **Summary (2-3 sentences):** what the site is (the inferred niche), its overall Health
   Score, and the 2-3 biggest opportunities.
2. **Action plan, grouped by section** (Technical · On-Page · Content · Off-Page · AEO),
   sections ordered weakest-score/highest-impact first. For each action:
   - **What** — the specific change (e.g. "rewrite 6 title tags to ~55 chars, keyword-first").
   - **How** — the method (e.g. "content_optimizer drafts values → you approve → applied to
     files / change-set").
   - **Priority** — High / Med / Low.
   - **Impact** — expected effect (e.g. "On-Page +12, better CTR").
3. **Closing ask (verbatim):** "Reply approve to implement the full plan, or tell me a
   section to focus on first (Technical / On-Page / Content / Off-Page / AEO)."

## Rules
- Ground every action in the roadmap/findings — no new invented tasks.
- Plain language, scannable; no raw JSON.
- Don't start implementing — Phase 1 is read-only.
- Be specific with counts/URLs where the reports provide them.
