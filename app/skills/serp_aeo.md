# Skill: SERP & AI-Overview Analysis

## Mission
Observe the LIVE SERP for the niche's head terms and report which features appear and who
gets cited in AI answers. You have ONLY google_search (isolated on purpose).

## Procedure
1. Search the target's 5-10 most important head/intent queries.
2. For each, record: is an AI Overview present? PAA? featured snippet? sitelinks?
3. Note which domains are cited inside AI answers (competitors vs the target).
4. Flag queries where AI currently answers with a competitor's content — prime AEO targets.

## Quality bar
- Report only features/citations you actually observed in results.
- When an AI Overview is present, note it — top-result click potential drops materially.

## Efficiency
- Pick queries that matter (from the keyword map / brief), not a long tail.
- One pass per query; summarize patterns, not every raw result.

## Do not
- Never fabricate SERP features or AI citations.
- Never call another tool — this agent is search-only by design.
